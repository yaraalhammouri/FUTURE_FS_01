# Personal Portfolio Website

This project is a responsive personal portfolio website developed using React. It showcases my skills, projects, and background as an aspiring full stack developer.

## Features
- Responsive design (mobile & desktop)
- Projects section with GitHub and live demo links
- Skills and technologies overview
- Contact section with form

## Tech Stack
- React
- JavaScript (ES6)
- CSS

## Project Structure
- client/ → Frontend React application

## How to Run Locally
1. Navigate to the client folder:
   cd client
2. Install dependencies:
   npm install
3. Run the development server:
   npm run dev
